The file contains:
chromosome=chromosome number
base_pair_location=base pair position on GRCh37/hg19
effect_allele_frequency=risk allele frequency
beta=effect size or log(OR)
standard_error=standard error
p_value=p-value
odds_ratio=odds ration
ci_lower=95% lower confidence interval
ci_upper=95% upper confidence interval
effect_allele=risk allele
other_allele=non risk allele
